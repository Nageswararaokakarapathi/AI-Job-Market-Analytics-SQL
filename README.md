# AI Job Market Analytics Using SQL

## Project Overview

This project analyzes a real-world AI Job Market dataset containing 10,345 job postings across multiple countries, industries, and job roles.

Using SQL, I answered 20 business-focused questions to uncover insights related to salaries, hiring trends, skills demand, experience levels, education impact, remote work opportunities, and industry hiring patterns.

The goal of this project is to demonstrate how SQL can transform raw job market data into actionable business intelligence.

---

## Dataset Information

* Records: 10,345
* Countries: USA, Canada, UK, Germany, India, Australia, Singapore
* Job Roles:

  * AI Engineer
  * Machine Learning Engineer
  * Data Scientist
  * Data Engineer
  * Data Analyst
  * Business Analyst

## Key Insights

* AI Engineers and Machine Learning Engineers receive the highest salaries.
* Senior professionals earn significantly more than entry-level candidates.
* Deep Learning and Machine Learning skills provide the largest salary boost.
* Canada and Singapore offer the highest average salaries.
* Hiring urgency strongly influences compensation.
* Remote, Hybrid, and Onsite jobs show very similar salary ranges.
